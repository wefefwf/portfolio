# dev.folio

풀스택 개발자 김도영의 개발 포트폴리오 사이트입니다.

## Tech Stack

`HTML` `CSS` `Vanilla JavaScript`

## Features

- **Art Circles Hero** — 좌우 원형 궤도에 아이템 배치, hover 시 중앙 카드 및 사이드 이미지 동기화 변경
- **프로젝트 모달** — 프로젝트 카드 클릭 시 모달 팝업 (영상 미리보기 + 상세 설명 + GitHub 링크)
- **원페이지 스크롤** — Hero / About / Values / Skills / Projects / Experience / Contact
- **스크롤 애니메이션** — Intersection Observer 기반 fade-up 효과
- **자동 순환** — Hero 섹션 프로젝트 4초 간격 자동 전환 (hover 시 정지)
- **반응형 디자인** — 데스크톱, 태블릿, 모바일 대응
